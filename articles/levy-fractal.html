Lévy's Fractal

Languages & Libraries

Background

I love watching programming related Youtube videos especially those provides by Daniel Shiffman on his channel The Coding Train. Through his tutorials, he teaches how to use the p5.js library by implementing some mathematical concepts.

Thinking that computer visualization is a really efficient way to introduce maths to newbies, I decided to make several programs that bring (I hope) a better understand of some notions. And I chose to start with the Lévy's fractal.

Lévy C curve, also called Lévy fractal, is a self-similar fractal curve first described in the 1900s. It bears the name of the French mathematician Paul Lévy who studied its self-similarity properties.

The construction of this fractal is very simple and can be done using an L-system. But there is a more interesting way to do it.

I watched a video produced by Code Parade called "Draw Fractals in Under 5 Minutes", where he explains how to quickly create fractal shapes using Microsoft Paint Tool. And when I tried to reproduce his tutorial by myself I noticed that never mind the starting shape, if I keep applying the same transfer and rotation I always ended with an approximation of a Lévy fractal.

So to verify that I created a Lévy fractal generator.

Description

The interface

Pixel editor

It composed of a 7x7 grid of squares where you can draw pixel shapes using mouse left-click.

When a shape is drawn two coloured square appears: a red one for the beginning and a green one for the end, to help keeping track of the successive rotations applied.

Rendering

When a shape is created with the pixel editor, you can click on the "Draw Fractal" button to generate the Lévy fractal. The slider next to it represented the number of iteration of the process (13 max to avoid lag).
